# 论文工作区

论文目录保存历史草稿和未来写作材料。当前 PyBullet 状态、实验门槛和研究协议以 [STATUS.md](../../STATUS.md) 与 [docs/projects/pybullet_td3/implementation_plan.md](../projects/pybullet_td3/implementation_plan.md) 为准。

`manuscript.md`、`method.md`、`results.md` 和 `outline.md` 是旧协议草稿，涉及 oracle 扰动观测或 PID-FF pilot，只能作为历史诊断，不能直接当作当前主结果。

新的论文文本必须在对应阶段证据完成后单独生成，并明确区分 MATLAB/Simulink 与 PyBullet 两条研究线。
