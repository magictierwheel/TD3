# Oracle/PID-FF pilot archive

This directory preserves the historical Oracle and PID-feedforward TD3 pilot exactly as historical evidence. The source files are not an active package and must not be imported by the current study. Reproduction requires checking out the original recorded Git commit and using its original layout, commands, and data paths.

The compact evidence files are under `evidence/`; the complete local runs remain outside Git under the corresponding ignored `runs/` directory when available.
